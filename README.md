## Integrate bflutter in dev mode

- Clone bflutter and place it in the same folder root with bflutter_poc
https://github.com/beesightsoft/bflutter

- Check dependencies pubspec.yaml of bflutter_poc
```
dependencies:
  ...
  bflutter:
    path: ../bflutter
```
